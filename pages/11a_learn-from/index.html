Learn gsap.from()

The gsap.from() method is the perfect tool for creating entrance animations. It animates an element *from* a set of specified values to its current, default state.

Instead of defining where the animation should end up, you define where it should start. This makes it incredibly intuitive for bringing elements into view in a stylish and engaging way.
