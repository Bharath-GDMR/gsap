Learn gsap.to()

The gsap.to() method is the most common and fundamental method in GSAP. It allows you to animate an element from its current state to a new, specified state.

Think of it as telling GSAP, "I want this element to go *to* these new values." You provide the target element and an object of properties you want to animate, and GSAP handles the rest.
