Gsap Set Challenge

Now that you know how to use gsap.set(), it's time for a challenge. Your task is to create a simple image gallery where the main image changes when you click on a thumbnail.

You'll use gsap.set() to instantly change the source of the main image, creating a seamless transition between images. This is a great way to practice your skills and create a common UI pattern.
