Learn gsap.set()

The gsap.set() method is a simple but powerful tool. It allows you to instantly set the properties of an element without any animation. This is useful for setting the initial state of an element before an animation begins, or for quickly changing its properties without a transition.

Think of it as a way to instantly teleport an element to a new state. It's a great way to set up your animations and ensure they start from the correct position.
